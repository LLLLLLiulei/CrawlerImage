# 如何安装
1. 命令行进入项目
2. pip install -r requirements.txt
3. 如果是windows，gevent安装不成功，<del>建议上 [http://www.lfd.uci.edu/~gohlke/pythonlibs/]() 直接下载编译好的安装</del>
4. 上面那个链接要翻墙，我下载好了放到InstallLib目录下了，自己根据32位64位系统选择安装

# 如何使用
1. 设置setting.py
2. 运行main.py
3. 爬取完毕后，运行browse.py，打开浏览器浏览
